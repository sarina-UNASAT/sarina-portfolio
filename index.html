01 // Identiteit

Over Mij

System & Network Engineering Student

Mijn naam is Sarina Chotoe en ik ben student aan UNASAT van cohort 2022. Ik volg de richting SNE, wat staat voor System and Network Engineering. Ik ben een gemotiveerde en nieuwsgierige student met interesse in technologie, creativiteit en persoonlijke ontwikkeling.

Tijdens mijn opleiding heb ik gewerkt met verschillende technologieën en tools zoals Linux, Visual Studio Code, Oracle VirtualBox, MySQL, HTML en CSS via schoolprojecten en praktische opdrachten. Ik ben erg leergierig, vooral wanneer ik goede begeleiding en samenwerking krijg.

Hoewel ik nog geen professionele IT-werkervaring heb, blijf ik mezelf ontwikkelen door middel van schoolprojecten, zelfstandig leren en praktijkervaring.
